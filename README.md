## Project Overview
Charis Technologies is a professional website designed to promote security and IT solutions for residential, business, and institutional clients. The site highlights the company’s services, portfolio, support options, and contact details in a clear and user-friendly format.

## Mission and Vision
- **Mission:** To deliver tailored security and IT solutions that empower clients with safety, efficiency, and confidence.
- **Vision:** To create smarter, safer environments by combining advanced technology with excellent service.

## Goals and Objectives
- Build trust and credibility through a professional online presence.
- Educate visitors about available services and solutions.
- Improve customer communication and support access.
- Showcase completed projects and success stories.
- Increase leads, inquiries, and overall business visibility.

## Target Audience
The website is intended for:
- Residential clients
- Small and medium businesses
- Corporate organisations
- Educational and institutional clients
- Property developers and event venues

## Core Website Features
- Home page with key service highlights and visuals
- About page with mission, vision, and company background
- Services page detailing CCTV, access control, alarm, and IT solutions
- Portfolio page showing completed work
- Support page with FAQs and assistance options
- Contact page with inquiry and quotation details
- Privacy Policy and Terms of Service pages

## Design Approach
- **Colour scheme:** deep blue, light blue, grey, white, and green
- **Typography:** modern sans-serif fonts for a clean layout
- **User experience:** responsive design, clear navigation, accessible content, and strong visual hierarchy

## Technical Requirements
- HTML
- CSS
- JavaScript
